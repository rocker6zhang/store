##项目目的
    java web 常用功能练习(不含框架)
##项目功能
* 用户管理
    *  用户注册
    *  用户登录
    *  用户注销
    *  用户激活(邮件)
* 商品管理
    *  添加商品（上传）
    *  商品查看-- 列表查询
    *  pdate&delete	
* 购物	
    *  商品列表
    *  商品详情
    *  添加购物车
    *  查看购物车
    *  修改购物车
    *  生成订单
    *  查看订单


##系统设计
* 技术选型
    *  JSP + Servlet + JavaBean + FileUpload + JavaMail
    *  C3P0 + MySQL
    *  Eclipse2017+ Tomcat7.0 + JDK6 + linux windows
    *  MVC 模式 
    *  JavaEE 三层结构
